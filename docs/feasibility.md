# 可行性分析：3DGS 用于文创手办建模

> 本文档诚实地分析 3D Gaussian Splatting 在文创产品建模中的能力边界。
> **核心结论：3DGS 非常适合视觉展示，但不适合工业级可打印 3D 模型。**

---

## 一、3DGS 的核心能力

### 擅长什么？

| 能力 | 说明 |
|------|------|
| **高保真视觉复刻** | 新视角合成质量可达照片级，PSNR > 30dB |
| **实时渲染** | 训练后 60+ FPS 交互式浏览 |
| **复杂外观** | 可处理半透明、毛发、薄结构等传统 mesh 难以表示的外观 |
| **光照效果** | 通过球谐函数捕捉视角依赖的颜色变化 |
| **少图适应** | 结合 FSGS/SparseGS adapter，20 张以内可获可用结果 |

### 不擅长什么？

| 限制 | 说明 |
|------|------|
| **几何精度** | 3DGS 是"基于点的视觉表示"，不是封闭网格 |
| **可打印性** | 直接导出的点云不能 3D 打印 |
| **水密性** | 没有封闭表面，没有体积概念 |
| **尺度真实性** | 默认无物理尺度，需要参照物标定 |
| **细节一致性** | 薄结构可能被多张不一致的照片"涂抹"掉 |

---

## 二、适用与不适用的物体

### ✅ 适合 3DGS 建模的物体

| 物体类型 | 原因 | 典型场景 |
|----------|------|----------|
| 哑光涂装手办 | 纹理丰富、无镜面反射 | 潮玩、动漫手办 |
| 陶瓷/陶器 | 表面微纹理、有机外形 | 手工杯、花瓶、茶具 |
| 木雕/竹编 | 丰富纹理、漫反射 | 文创摆件、手工饰品 |
| 树脂/塑料模型 | 哑光表面、多色印刷 | 盲盒、积木、模型套件 |
| 织物玩偶 | 丰富纹理、无硬边缘 | 布艺手工、毛绒玩具 |
| 皮革制品 | 表面纹理适合特征匹配 | 手工皮具、钱包 |

### ❌ 不适合 3DGS 建模的物体

| 物体类型 | 原因 | 替代方案 |
|----------|------|----------|
| 镜面/镀铬物体 | COLMAP 特征匹配完全失效 | 需特殊采集方案（交叉偏振光） |
| 透明玻璃/水晶 | 折射和反射干扰重建 | 传统摄影测量 + 表面喷涂 |
| 极细结构（<1mm） | 低于照片分辨率 | 微距镜头 + 更多照片 |
| 单一纯色无纹理 | 无特征点可匹配 | 投影纹理或表面喷涂 |
| 自我遮挡严重的物体 | 部分表面始终不可见 | 增加视角或使用镜面辅助 |
| 动态/变形物体 | 违背 SfM 的静态假设 | 需特殊方法（非刚性 NeRF） |

## 三、mesh 导出（Beta 功能）的诚实说明

### 当前状态

CraftSplat-20 的 `--formats glb` 导出为 **Beta 功能**。3DGS 本身不能直接生成可用的封闭网格。

### 可行路径

| 方法 | 原理 | 质量 |
|------|------|------|
| **SuGaR** | 在 3DGS 上施加表面对齐约束 | ⭐⭐⭐ 较好 |
| **2DGS** | 使用 2D Gaussian 替代 3D，天然更适合表面提取 | ⭐⭐⭐ 较好 |
| **GS2Mesh** | 从训练好的 3DGS 使用 TSDF 融合提取 mesh | ⭐⭐ 一般 |
| **Poisson 重建** | 从 Gaussian 中心点云直接 Poisson 重建 | ⭐ 较差 |

### 预期效果

- **可以做到**: 生成视觉上"像"物体形状的 mesh
- **不能做到**: 生成可直接 3D 打印的水密 STL
- **差距**: 需要在 Blender/ZBrush 中手动修复（补洞、平滑、重新拓扑）

## 四、与其他方案的对比

| 方案 | 照片数 | 输出 | 视觉质量 | 几何质量 | 成本 |
|------|--------|------|----------|----------|------|
| **CraftSplat-20 (3DGS)** | 18-20 | 点云 + 新视角渲染 | ⭐⭐⭐⭐⭐ | ⭐⭐ | 低 |
| Photogrammetry (Metashape) | 50-200 | 纹理 mesh | ⭐⭐⭐ | ⭐⭐⭐⭐ | 中 |
| 激光扫描 | N/A | 高精度点云 | N/A | ⭐⭐⭐⭐⭐ | 高 |
| NeRF (Nerfacto) | 20-100 | 体积渲染 | ⭐⭐⭐⭐ | ⭐ | 低 |
| 手动 3D 建模 | 0-5 (参考) | 面片 mesh | ⭐⭐⭐ | ⭐⭐⭐⭐ | 人工成本 |

## 五、总结

**CraftSplat-20 适合的场景：**

- 🎨 文创产品在线展示（3D 产品查看器）
- 🏪 电商 3D 预览
- 📱 社交媒体 AR 滤镜素材
- 🎓 计算机视觉课程设计/毕业设计
- 🔬 3DGS 研究和实验

**CraftSplat-20 不适合的场景：**

- 🖨️ 直接 3D 打印
- 🏭 工业逆向工程
- 📐 精密测量
- 💎 珠宝级细节复刻
