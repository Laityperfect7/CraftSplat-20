# 18+2 张拍摄指南

> CraftSplat-20 推荐拍摄协议：**18 张环绕 + 2 张校准 = 20 张以内完成高质量三维重建**

---

## 为什么是 20 张？

传统的 3D Gaussian Splatting 方法（如原版 3DGS）通常需要 50-200 张照片。CraftSplat-20 针对文创手办场景优化了拍摄协议和参数配置，使得在理想拍摄条件下，**18+2 张照片即可获得可用的视觉重建结果**。

核心原理：
- 手办/摆件体积小（通常 5-30cm）、表面纹理丰富
- 合理的环绕角度分布 + Nerfstudio Splatfacto 的少图优化
- 可选 FSGS/SparseGS adapter 进一步增强少图质量

---

## 拍摄设备

| 设备 | 建议 | 备注 |
|------|------|------|
| **相机** | 手机主摄或微单 | 等效 24-50mm 焦距 |
| **三脚架** | 推荐 | 手持也可，但需稳定 |
| **转盘** | 强烈推荐 | 手动或电动均可 |
| **灯光** | 柔光箱 / LED 面板 | 避免强烈单侧阴影 |
| **背景** | 纯色无缝纸 | 灰/白色最佳，黑色次之 |
| **标尺** | 可选 | 用于尺度还原 |

## 18+2 拍摄方案

### 环 1：上环（高角度环绕）— 8 张

```
仰角: +30°（相机略高于物体水平面，向下俯拍）
数量: 8 张
间隔: 每 45° 一张
```

| # | 方位角 |
|---|--------|
| 1 | 0° |
| 2 | 45° |
| 3 | 90° |
| 4 | 135° |
| 5 | 180° |
| 6 | 225° |
| 7 | 270° |
| 8 | 315° |

### 环 2：中环（水平环绕）— 6 张

```
仰角: 0°（相机与物体中心等高）
数量: 6 张
间隔: 每 60° 一张，与上环错开 15°
```

### 环 3：下环（低角度环绕）— 4 张

```
仰角: -20°（相机略低于物体，向上仰拍）
数量: 4 张
间隔: 每 90° 一张，与中环错开 15°
```

### 校准视角 — 2 张

```
仰角: +55° 和 +60°
用途: 用于尺度/坐标系标定，不参与训练
```

### 总计

```
上环 8 + 中环 6 + 下环 4 + 校准 2 = 20 张
训练用: 18 张
校准用: 2 张
```

## 拍摄参数设置

### 相机设置

| 参数 | 建议值 | 说明 |
|------|--------|------|
| **ISO** | 100-400 | 尽量低，减少噪点 |
| **光圈** | f/5.6 - f/8 | 保证足够景深 |
| **快门** | 1/60 - 1/200 | 手持时 1/125 以上 |
| **白平衡** | 固定 | 不要用自动白平衡 |
| **对焦** | 手动 | 锁定对焦点在物体上 |
| **分辨率** | ≥12MP | 4000×3000 或更高 |

### 拍摄环境

- 🚫 **禁止闪光灯**：强光会让 COLMAP 特征匹配失败
- 🚫 **禁止自动曝光**：每张照片曝光一致
- ✅ **柔和均匀光**：使用柔光箱或阴天窗光
- ✅ **固定光源**：不要在不同照片之间移动灯光
- ✅ **物体不动**：如果是转盘方案，确保只有转盘旋转，物体在转盘上不滑动

## 常见错误

### ❌ 错误 1：照片间光照不一致

**现象**: 自动曝光或移动光源导致某些照片过亮/过暗。
**解决**: 使用手动曝光模式，锁定所有参数。

### ❌ 错误 2：物体在照片间发生了移动

**现象**: COLMAP 稀疏重建失败或点云非常稀疏。
**解决**: 使用转盘，确保物体粘牢在转盘上。

### ❌ 错误 3：浅景深导致部分区域模糊

**现象**: 大光圈（如 f/1.8）下只有物体前部对焦清晰。
**解决**: 缩小光圈到 f/5.6 或 f/8，或在后期合成景深堆叠。

### ❌ 错误 4：镜面反光

**现象**: 光泽表面在 COLMAP 中产生错误的特征匹配。
**解决**: 使用偏光镜，或调整光源角度消除反光。

### ❌ 错误 5：背景太杂乱

**现象**: COLMAP 在背景纹理上产生大量匹配点，干扰物体重建。
**解决**: 使用纯色背景纸/布，或对每张照片生成 mask。
